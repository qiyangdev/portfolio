我会在 Apple 平台、移动端和 Web 上构建产品，而每一种界面都需要不同的打磨方式。

SwiftUI 是我塑造 Apple 原生体验的地方。Expo 让移动端的迭代足够快，同时不失去覆盖面。Next.js 则是我默认的 Web 方案。

目标不是收集更多框架，而是选对场景、做出有用的产品，并在工程路上持续学习。

```ts
const stack = ["SwiftUI", "Expo", "Next.js"] as const
```
